# space-object-classification-ml
Machine learning project to classify astronomical objects (stars, galaxies) using SDSS data with multiple models and evaluation techniques.

Objective
To build and evaluate machine learning models that can accurately classify space objects based on their photometric properties.

📊 Dataset
Source: Sloan Digital Sky Survey (SDSS)
Features include:
Spatial coordinates (ra, dec)
Photometric magnitudes (modelMag_u, g, r, i, z)
PSF magnitude (psfMag_r)
Target:
object_type → STAR / GALAXY
